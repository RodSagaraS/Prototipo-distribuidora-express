# Distribuidora Express

## Descripción

Prototipo no funcional de una aplicación móvil para una empresa de distribución de alimentos. La app permite iniciar sesión con Gmail, revisar productos, agregarlos al carrito, calcular automáticamente el valor del despacho y consultar el estado del pedido.

## Reglas de negocio

* Compras sobre $50.000 dentro de 20 km: despacho gratis.
* Compras entre $25.000 y $49.999: $150 por kilómetro recorrido.
* Compras menores a $25.000: $300 por kilómetro recorrido.
* Carnes y mariscos congelados requieren monitoreo de temperatura.
* Si la temperatura del congelador del camión supera el límite permitido, se emite una alerta en el dispositivo móvil.

## Pantallas del prototipo

1. Inicio
2. Login con Gmail
3. Catálogo de productos
4. Carrito y cálculo de despacho
5. Confirmación de compra
6. Estado del despacho
7. Monitoreo de temperatura
8. Menú administrador

## Herramienta utilizada

El prototipo fue diseñado utilizando Pencil Project como referencia de prototipado no funcional.
